# Requisitos

##### Exercicio (ex1.js)

Faça cinco programas, um para cada operação aritmética básica. 
Seu programa deve ter duas variáveis a e b definidas no começo com os valores que serão operados. 

Faça programas para:
* Adição
* Subtracão
* Multiplicação
* Divisão
* Módulo
