1 - npm install redux-thunk

2 - Para habilitar o uso dele na sua aplicação, é preciso fazer uso da função applyMiddleware() do Redux :


// arquivo onde a redux store é criada:

import { createStore, applyMiddleWare } from 'redux';
import thunk from 'redux-thunk';
import reducer from '/path/to/your/root/reducer';

...

const store = createStore(reducer, applyMiddleware(thunk));
...

3 - thunk nada mais é do que uma função que encapsula uma operação para que ela seja feita posteriormente. Em termos práticos, isso significa que você está definindo uma função que vai ser retornada por uma outra função com mais lógica adicionada a ela.
